Tutorial - Step 1: Introduction

Welcome to the in-simulation tutorial of Mars Simulation Project.

Through these steps you will become familiar with the basic tools and viewers and learn how to overview and interact with the simulation. The tutorial has 12 steps, mostly focused on the various tool windows. Expect it to take around 10-15 minutes.

The tutorial is supplemented by a more section-oriented guide, also accessible from the Help menu.

You can close this tutorial window at any time and reopen it from the Help menu.
