resources.xml

Resources are materials that are used and stored in the simulation. Resources are measured in kg.

The resources.xml configuration file contains configuration for all the resources used in the simulation.

A "resource" element has a unique "name" attribute.

It has a required "phase" attribute that represents the phase of matter which it is stored and used in the simulation. Possible values are "solid", "liquid" and "gas".

It has a required "life-support" attribute represents if the resource is used for life support purposes or not. Possible values are "true" and "false".
