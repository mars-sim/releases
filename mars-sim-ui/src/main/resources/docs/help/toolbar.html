Tool Bar

The buttons on the tool bar perform the following actions:

File Buttons

  • New... - Creates and starts a new simulation. Opens the Simulation Configuration Editor dialog.
  • Open... - Loads a simulation from a save file.
  • Save - Saves the current simulation to the default save file.
  • Save As... - Saves the current simulation to a save file.

Tool Buttons

  • Mars Navigator Tool - Opens the Mars Navigator tool.
  • Search Tool - Opens the Search tool.
  • Time Tool - Opens the Time tool.
  • Monitor Tool - Opens the Monitor tool.
  • Mission Tool - Opens the Mission tool.
  • Settlement Map Tool - Opens the Settlement Map tool.
  • Science Tool - Opens the Science tool.
  • Resupply Tool - Opens the Resupply tool.

Other Buttons

  • User Guide - Opens this User Guide window.
