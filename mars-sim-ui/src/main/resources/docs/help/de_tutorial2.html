Tutorium - Schritt 2: Das Hauptfenster der Simulation

Das Hauptfenster ist das große Fenster, in dem sich die Simulation abspielt. Es hat einen rötlichen Hintergrund, eine Menüleiste, eine Werkzeugleiste und unten eine Einheitenleiste, die geöffnete Einheiten auflistet - anfangs noch keine. Weiters können hier allerlei Werkzeugfenster erscheinen.

Beim Start einer neuen Simulation wird ein Fenster schon geöffnet: der Mars-Navigator. Mehr dazu in Schritt 4, nach einer Betrachtung der Menüs.
